# Урок 5. Вероятности и анализ риска

## 1. Цель занятия

Освоить вероятностный подход к принятию финансовых решений и научиться:

- рассчитывать вероятности событий;
- использовать математическое ожидание;
- измерять риск через дисперсию и стандартное отклонение;
- применять нормальное распределение;
- оценивать вероятность убытка;
- сравнивать проекты по соотношению доходности и риска.

## 2. Базовые понятия теории вероятностей

### 2.1 Вероятность события

P(A) = число благоприятных исходов / общее число исходов  

`0 ≤ P(A) ≤ 1`

Если P(A) = 0 → событие невозможно  
Если P(A) = 1 → событие достоверно  

### 2.2 Сложение вероятностей

Если события несовместны:

`P(A ∪ B) = P(A) + P(B)`

Если события совместны:

`P(A ∪ B) = P(A) + P(B) − P(A ∩ B)`

### 2.3 Условная вероятность

`P(A | B) = P(A ∩ B) / P(B)`

Используется при зависимых событиях (например, вероятность дефолта при ухудшении рынка).

## 3. Случайная величина

Случайная величина X – числовой результат случайного события.

Пример: прибыль проекта.

Если есть k сценариев:

X принимает значения x1, x2, …, xk  
С вероятностями p1, p2, …, pk  

## 4. Математическое ожидание

Формула:

`E(X) = Σ (xi * pi)`

Показывает средневзвешенный результат.

### Excel

Если прибыли в B2:B4, вероятности в C2:C4:

`=СУММПРОИЗВ(B2:B4;C2:C4)`

## 5. Дисперсия и риск

### 5.1 Дисперсия

`Var(X) = Σ (xi − E(X))² * pi`

### 5.2 Стандартное отклонение

`σ = √Var(X)`

Чем выше σ, тем выше неопределенность результата.

## 6. Коэффициент вариации

`CV = σ / E(X)`

Позволяет сравнивать риск проектов разного масштаба.

Если:
- CV меньше → проект относительно менее рискованный;
- CV больше → риск выше относительно ожидаемой доходности.

## 7. Нормальное распределение

Предполагаем, что показатель распределён нормально.

Плотность (идея):

`f(x) = (1 / (σ√(2π))) * e^(-(x − μ)² / (2σ²))`

В Excel:

Вероятность, что X ≤ x:
`=НОРМ.РАСП(x; μ; σ; ИСТИНА)`

Обратная функция (квантиль):
`=НОРМ.ОБР(p; μ; σ)`

## 8. Вероятность убытка

Если μ – средняя прибыль, σ – стандартное отклонение.

Вероятность того, что прибыль < 0:
`=НОРМ.РАСП(0; μ; σ; ИСТИНА)`

Если результат 0,12 → 12% вероятность убытка.
