# Урок 3. Описательная статистика и анализ данных

## 1. Цель занятия

Освоить методы количественного анализа данных и научиться:
- рассчитывать основные статистические показатели;
- интерпретировать структуру распределения;
- выявлять выбросы;
- анализировать взаимосвязь показателей;
- оценивать устойчивость финансовых метрик.

## 2. Типы данных

### 2.1 Количественные
- дискретные (N клиентов);
- непрерывные (выручка, прибыль, маржа).

### 2.2 Качественные
- номинальные (регион);
- порядковые (уровень риска).

## 3. Меры центральной тенденции

### 3.1 Среднее арифметическое

```
x̄ = (1/n) * Σ xi
```

Чувствительно к выбросам.

### 3.2 Взвешенное среднее
```
x̄w = (Σ wi * xi) / (Σ wi)
```
Используется при различной значимости наблюдений (например, средняя цена с учетом объема продаж).

### 3.3 Медиана

Если n нечётное:  
```
Me = x((n + 1) / 2)
```
Если n чётное:  
```
Me = (x(n / 2) + x(n / 2 + 1)) / 2
```

### 3.4 Мода
Наиболее часто встречающееся значение.

## 4. Меры вариации

### 4.1 Размах
```
R = xmax − xmin
```
### 4.2 Дисперсия

Выборочная:
```
s² = Σ(xi − x̄)² / (n − 1)
```
Генеральная:
```
σ² = Σ(xi − μ)² / n
```
### 4.3 Стандартное отклонение
```
s = √(s²)
```
Показывает средний масштаб отклонения.

### 4.4 Коэффициент вариации
```
CV = (s / x̄) * 100%
```
Позволяет сравнивать нестабильность показателей разного масштаба.

## 5. Квартильный анализ

### 5.1 Квартиль

- Q1 — 25% наблюдений ниже;
- Q2 — медиана;
- Q3 — 75% наблюдений ниже.

### 5.2 Межквартильный размах
```
IQR = Q3 − Q1
```
## 6. Выбросы

Нижняя граница:
```
Q1 − 1.5 * IQR
```
Верхняя граница:
```
Q3 + 1.5 * IQR
```
## 7. Асимметрия и эксцесс

### 7.1 Коэффициент асимметрии
```
Skew = (1/n) * Σ((xi − x̄) / s)³
```
- Skew > 0 — правосторонняя асимметрия;
- Skew < 0 — левосторонняя.

### 7.2 Эксцесс (крутизна)
```
Kurt = (1/n) * Σ((xi − x̄) / s)⁴ − 3
```
Показывает «тяжесть хвостов».

## 8. Нормальное распределение

Плотность:
```
f(x) = (1 / (σ * √(2π))) * e^(-(x − μ)² / (2σ²))
```
Правило 68–95–99.7:
- 68% значений в пределах ±1σ;
- 95% — ±2σ;
- 99.7% — ±3σ.

## 9. Ковариация и корреляция

### 9.1 Ковариация
```
cov(x,y) = Σ((xi − x̄)(yi − ȳ)) / (n − 1)
```
### 9.2 Коэффициент корреляции Пирсона
```
r = cov(x,y) / (sx * sy)
```
Где:
- r ∈ [-1; 1].

## 10. Линейная регрессия (базовая модель)

Модель:
```
y = a + b*x
```
Оценка коэффициента наклона:
```
b = cov(x,y) / sx²
```
Свободный член:
```
a = ȳ − b*x̄
```

Коэффициент детерминации:

R² = r²
